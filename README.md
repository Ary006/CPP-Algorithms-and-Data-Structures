# C++ Algorithms and Data Structures

A collection of C++ implementations covering classic 
algorithms, data structures and constraint solving. 
Part of a 94/100 scored programming module at 
King's College London.

## Parts

**Part 1 — Countdown Solver**
Recursive solver for the Countdown numbers game. 
Combines arithmetic operations across a set of 
numbers to reach a target value, with infix-to-RPN 
expression conversion and stack-based evaluation.

**Part 2 — String Construction**
Cost-minimising string builder using the Z-function 
algorithm. Determines the optimal combination of 
character append and substring clone operations to 
construct a target string.

**Part 3 — Doubly Linked List**
Full templated doubly linked list implementation 
with head/tail pointers, bidirectional iterators, 
insert, erase, reverse, push_front and push_back.

**Part 4 — Set Puzzle Solver**
Backtracking constraint-satisfaction solver for a 
Sudoku-variant number placement puzzle, using 
minimum-remaining-values heuristic to optimise 
cell selection.

## Stack
C++ — header-only implementations, no external 
dependencies
